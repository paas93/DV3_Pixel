# PixelTileConverter

이미지를 지정된 인게임 타일 색상 팔레트로 변환하는 브라우저용 도구입니다.

## 사용 방법
1. `index.html`을 브라우저에서 엽니다.
2. 이미지를 업로드합니다.
3. 프리셋 또는 커스텀으로 그리드 가로/세로 칸 수를 지정합니다.
4. `변환하기`를 누릅니다.
5. 결과 위에 마우스를 올리면 타일명, HEX, 좌표가 표시됩니다.
6. PNG 또는 CSV로 결과를 다운로드할 수 있습니다.

## 주요 기능
- 사용자가 지정한 47개 타일 팔레트 기준 색상 변환
- 16×16부터 192×192까지 대표 프리셋 제공
- 최대 300×300 커스텀 그리드 지원
- 가로/세로가 정방형이 아닌 커스텀 그리드에서도 원본 이미지 비율 유지
- 남는 영역은 `빈 칸`으로 처리
- 그리드 라인 표시/숨김
- 타일명 약어 표시
- 디더링 옵션
- PNG/CSV 다운로드

## 파일 구성
- `index.html` : 실행 파일
- `styles.css` : 화면 스타일
- `app.js` : 변환 로직
